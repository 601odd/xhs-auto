# 小红书自动发帖工具 🚀

基于浏览器MCP和AI技术的小红书自动化发帖工具，支持智能内容生成、定时发布和反检测机制。

## ✨ 功能特点

- 🤖 **智能内容生成**: 集成OpenAI API，自动生成高质量的内推内容
- 🕒 **定时发布**: 支持自定义时间间隔和cron表达式定时发布
- 🛡️ **反检测机制**: 内置人性化操作模拟，绕过平台自动化检测
- 📱 **多平台支持**: 基于Puppeteer，支持多种浏览器环境
- 📊 **发布统计**: 实时监控发布状态和每日发布限制
- 🎯 **内推专精**: 专门优化的内推内容生成和发布流程

## 🛠️ 技术栈

- **Node.js** - 后端运行环境
- **Puppeteer** - 浏览器自动化
- **OpenAI API** - AI内容生成
- **node-cron** - 定时任务调度
- **ES6 Modules** - 现代JavaScript模块

## 📋 系统要求

- Node.js >= 16.0.0
- Chrome/Chromium 浏览器
- 稳定的网络连接

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env` 并填入配置：

```bash
cp .env.example .env
```

编辑 `.env` 文件：

```env
# 小红书账号配置
XHS_USERNAME=你的手机号或邮箱
XHS_PASSWORD=你的密码
XHS_REFERRAL_CODE=你的内推码

# OpenAI API配置
OPENAI_API_KEY=你的OpenAI_API密钥
OPENAI_BASE_URL=https://api.openai.com/v1

# 浏览器配置
HEADLESS=false
SAVE_SCREENSHOTS=true

# 发帖配置
POST_INTERVAL_MIN=60
POST_INTERVAL_MAX=180
DAILY_POST_LIMIT=5
```

### 3. 启动程序

```bash
npm start
```

## 📖 使用指南

### 基本操作

1. **登录小红书**: 首次使用需要手动登录验证
2. **手动发帖**: 立即发布一条内推信息
3. **启动自动发帖**: 根据配置的时间间隔自动发布
4. **查看状态**: 监控当前运行状态和发布统计

### 自定义内容

程序支持两种内容生成方式：

#### AI生成内容
- 基于OpenAI API自动生成专业的内推内容
- 自动优化标题、正文和话题标签
- 根据当前热点调整内容风格

#### 模板内容
- 内置精心设计的内推模板
- 自动插入配置的内推码
- 确保内容质量和一致性

### 定时发布

支持多种定时发布方式：

```bash
# 每天9点、15点、21点发布
0 9,15,21 * * *

# 每小时发布一次
0 * * * *

# 工作日上午10点发布
0 10 * * 1-5
```

## ⚙️ 配置说明

### 发帖配置

- `POST_INTERVAL_MIN/MAX`: 自动发帖的时间间隔范围（分钟）
- `DAILY_POST_LIMIT`: 每日发帖数量限制
- `USE_STEALTH`: 是否启用反检测模式

### 反检测机制

- **随机延迟**: 模拟真实用户的操作间隔
- **人性化点击**: 随机化鼠标移动和点击位置
- **智能等待**: 根据页面加载情况调整等待时间
- **Cookie管理**: 自动保存和加载登录状态

3. **内容质量**: 确保发布内容符合平台规范

## ❓ 常见问题

**Q: 登录失败怎么办？**
A: 检查网络连接，清除浏览器缓存，或手动完成验证码验证。

**Q: 为什么发帖失败？**
A: 可能是网络问题、账号限制或页面结构变化，查看日志获取详细信息。

**Q: 如何自定义发帖内容？**
A: 修改OpenAI提示词或使用自定义内容发布功能。

**Q: 程序会被检测吗？**
A: 内置多重反检测机制，但建议适度使用，避免过于频繁。
